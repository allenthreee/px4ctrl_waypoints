**核心策略**  
- 每天 2-3 小时：1 小时学习/复习思路，1-2 小时编码实践

## 补充建议

- **困难题**（如 42 接雨水、76 最小覆盖子串）：一次做不出来很正常，先看题解理解思路，一周后再独立写一遍。
- **动态规划部分**（72 编辑距离、5 最长回文子串等）未排入本 30 天计划，建议之后专门花 5-7 天攻克。
- **每天记录**：用 Excel 或笔记本标记完成状态：✅ 独立写出 / ⚠️ 看提示后写出 / ❌ 不会（复习日重点攻克）。
